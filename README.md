# ✈️ Yatra Junction

> **One Stop for Every Yatra**

Yatra Junction is a modern travel booking platform designed to provide a seamless travel experience through an elegant and responsive web interface. The platform allows users to explore and book flights, trains, hotels, cabs, rentals, and curated holiday packages from a single destination.

## 🌍 Live Demo

**Website:** https://abhinandan-2903.github.io/Yatra-Junction/

## ✨ Features

* ✈️ Flight Booking
* 🚆 Train Booking
* 🏨 Hotel Booking
* 🚖 Cab Booking
* 🚗 Car Rentals
* 🏝️ Holiday Packages
* 👤 User Authentication
* 📋 Booking Management
* 🎟️ Package Details
* 📱 Fully Responsive Design
* 🌙 Premium Modern UI
